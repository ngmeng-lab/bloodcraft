Welcome to the open-source repository of Bloodcraft!
All of the code I use is available here, including some in-code comments.

Bloodcraft is a fantasy-themed mod centered around sacrifice and blood mechanics. By consuming your own health and sacrificing other creatures, you can achieve semi-permanent or permanent enhancements to weapons—especially scythe-type weapons.

The mod is currently under active development and is being gradually expanded and improved.
